# Asok75 Case
A simple acrylic sandwich case for the pcb.
Layer thicknesses:
top.dxf - 3mm
tophex.dxf - 4.5mm
plate.dxf - 5mm
mid.dxf - 3mm x 2
bottom.dxf - 3mm
foot1.dxf - 3mm
foot2.dxf - 3mm
foot3.dxf - 3mm
foot4.dxf - 3mm

Use 15-16mm length and 8-9mm length, 5mm flat to flat hex standoffs and M3 screws.
